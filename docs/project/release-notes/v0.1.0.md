# Mercator v0.1.0

Mercator `v0.1.0` is the first public, pre-GA evaluation release of the
event-sourced OCI run broker. It is intended for local evaluation, design
review, and early integration experiments, not production-critical workload
dispatch.

Start with the fake adapter unless you need to prove Docker or RunPod adapter
behavior. The fake path exercises the broker lifecycle without Docker, provider
credentials, registry access, or billable compute.

## What's Included

- Single-process Go server with embedded SQLite event log.
- JSON HTTP API, OpenAPI description, and JSON-first CLI.
- Embedded operator console for runs, placement decisions, offers, connections,
  workloads, and sink replay.
- Fake adapter for deterministic local evaluation.
- Docker-oriented local adapter path and RunPod-oriented provider path.
- Hand-written TypeScript, Python, and Ruby SDKs, installable from a source
  checkout.
- CI-backed fake-adapter smoke test and release archive builder.

## Install

Download the archive for your OS and architecture from the GitHub Release,
then download `checksums.txt` from the same release.

For Linux:

```sh
version=v0.1.0
os=linux
arch=amd64

curl -LO "https://github.com/benngarcia/mercator/releases/download/${version}/mercator_${version}_${os}_${arch}.tar.gz"
curl -LO "https://github.com/benngarcia/mercator/releases/download/${version}/checksums.txt"
sha256sum -c checksums.txt --ignore-missing
tar -xzf "mercator_${version}_${os}_${arch}.tar.gz"
sudo install "mercator_${version}_${os}_${arch}/mercator" /usr/local/bin/mercator
```

For macOS, set `os=darwin`, choose `arch=arm64` or `amd64`, and use:

```sh
shasum -a 256 -c checksums.txt --ignore-missing
```

Each archive contains `mercator`, `README.md`, `LICENSE`, and `NOTICE`.

## First Evaluation

From a source checkout, run:

```sh
scripts/smoke-test-fake.sh
```

Expected result: a fake-adapter run closes with `outcome=succeeded`,
`exit_code=0`, `cleanup=confirmed`, and `closed=true`.

For the full walkthrough, use:

- `README.md`
- `docs/production/fake-eval-path.md`
- `docs/reference/cli.md`
- `docs/reference/openapi.md`

## SDKs

SDK registry publishing is intentionally deferred for `v0.1.0`. Install SDKs
from a source checkout until package names, registry ownership, provenance, and
clean-environment install tests are confirmed.

See:

- `sdk/typescript/README.md`
- `sdk/python/README.md`
- `sdk/ruby/README.md`

## Known Limitations

Read `docs/production/known-limitations.md` before relying on this release for
anything production-adjacent. Important pre-GA gaps include package registry
publishing, public social proof, stronger external sink wiring, registry
credential flows, and external threat-model review.

Security issues should be reported privately through `SECURITY.md`.
